# Market Aggregator

A unified API for prediction markets (Polymarket, Limitless, Opinion, Kalshi) powered by `@alango/dr-manhattan`.

## Getting Started

1. Install dependencies:
   ```bash
   pnpm install
   ```

2. Start the development server:
   ```bash
   pnpm dev
   ```

## API Endpoints

- `GET /api/markets` - Fetch all markets from supported platforms.
- `GET /health` - Health check.

## Supported Platforms

- Polymarket
- Limitless
- Opinion
- Kalshi
